配置

进入工作目录会发现有一个_config.yml的文件，配置就是在这个文件中添加和修改的。

这里简单介绍一下几个常用的参数：

参数名 描述
title 网站标题
subtitle 网站副标题
description 网站描述
author 你的名字
language 网站使用的语言，例如zh-Hans就是指中文
timezone 网站时区，例如Asia/Shanghai就是指上海时区
theme 网站主题，使用之前需要先安装
avatar 你的头像
favicon 网页tag的小图标

这些知识简单介绍一下，具体的配置说明详见Hexo官方配置说明
